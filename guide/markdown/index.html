# Markdown 新语法

vuepress-theme-hope 通过内置 md-enhance，在 Markdown 中启用了更多的语法与新功能。

.vuepress/config.js 中的 themeconfig.markdown 会直接传递给插件作为插件选项。所以你也可以直接阅读 md-enhance 文档 查看用法。

提示

请不用担心你的网站大小，如果你不启用相关功能，最终代码不会包含这些功能相关的代码。

同时 vuepress 本身新增了一些 Markdown 语法，如果你对它们不熟悉，可以 查看官方文档。

# 一键启用

你可以设置 themeconfig.markdown.enableAll 启用 md-enhance 插件的所有功能。



 
 
 



module.exports = {
  themeConfig: {
    markdown: {
      enableAll: true
    }
  }
};
