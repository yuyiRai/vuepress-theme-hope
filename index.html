vuepress-theme-hope

一个轻量的 Vuepress 主题

快速上手 💡 配置 🛠

主题色切换

支持自定义主题色并允许用户在预设的主题颜色之间切换

深色模式

可以自由切换浅色模式与深色模式

浏览量与评论

配合 Valine 与 Vssue 来开启阅读量统计与评论支持

文章信息展示

为你的文章添加作者、写作日期、预计阅读时间、字数统计等信息

Markdown 增强

新增文字对齐、上下角标、脚注、标记、公式与流程图支持

博客支持

为你的文章添加日期、标签和分类，将自动生成文章、分类、标签与时间轴列表

文章加密

你可以为你的特定页面或特定目录进行加密，以便陌生人不能随意访问它们

更多新特性

包括博客主页、图标支持、路径导航、页脚支持、全屏按钮、返回顶部按钮等

PWA 支持

内置的 PWA 插件会让你的网站更像一个 APP。

TS 支持

为你的 Vuepress 开启 Typescript 支持

# 使用主题

你可以直接使用 vuepress-theme-hope 模板 来开始你的 vuepress 之旅。

# 安装

npm i -D vuepress-theme-hope
1

# 使用

// .vuepress/config.js
const resolve = require("vuepress-theme-hope/resolve");

module.exports = resolve({
  // your config here
});

提示

引入 resolve 函数的作用是在你编辑配置的时候，通过 TS 的 Interface 以及 JSDoc，很方便的给你提示。

同时，resolve 函数也会为你补全一些直接交付给 vuepress 的默认配置。
